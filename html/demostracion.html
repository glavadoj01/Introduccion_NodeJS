API Local de libros

Introducción

Esta página demuestra cómo utilizar la API local de libros para gestionar una colección de libros.

Funciones Externas

  • GET /api/libros/:genero: Obtiene una lista de libros relacionados con la tematica (generos similares, contiene titutlo, ...).
  • POST /api/persistir/:genero: Misma consulta que GET, pero guardando resultados en BD Local.

Funciones principales de la BD Local

  • GET /: Ruta de confirmación de funcionamiento y muestra de rutas
  • POST /libro: Agrega un nuevo libro
  • PUT /libro/:id: Actualizar información de un libro
  • DELETE /libro/:id: Eliminar un libro existente
  • GET /catalogo: Listar todos los libros disponibles
  • GET /libro?filtro=valor: Listar con filtros por titulo, autor y/o genero
  • USE invalid URL: Redirige a Página de Confirmación con indicación de error en Ruta
  • GET /demo: Esta pagina demostrativa
